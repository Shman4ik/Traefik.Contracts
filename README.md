# Traefik.Contracts
Библиотека с сущностями на .net которые используются в [Traefik Proxy](https://traefik.io/traefik/).

Помогает реализовать [HTTP](https://doc.traefik.io/traefik/providers/http/) провайдер для .Net сервисов.
